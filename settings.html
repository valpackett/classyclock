Backup Restore

To backup: tap Backup, copy the text, paste it somewhere (in your favorite notes app).
To restore: copy the text of your backup, paste it above, tap Restore.

Import from My Class Schedule

Make sure you have a file manager such as Cabinet or Sliding Explorer installed on your device.
Go to My Class Schedule → Settings → Backup / Restore → Create new backup.
Tap the file chooser here, select your file manager, go to your SD Card → My Class Schedule and choose the latest backup file. Finally, tap Import.
